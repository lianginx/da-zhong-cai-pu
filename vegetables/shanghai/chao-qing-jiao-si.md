# 炒青椒丝

## 原料

| 名称   | 重量   | 名称 | 重量  |
| ------ | ------ | ---- | ----- |
| 柿子椒 | 300 克 | 精盐 | 适量  |
| 红辣椒 | 2 个   | 食油 | 30 克 |
| 白糖   | 少许   | 味精 | 少许  |

## 制作方法

1. 将红辣椒去蒂和籽，洗净，切成细丝。
2. 将柿子椒去蒂和籽，洗净，切成细丝。
3. 将炒锅置火上烧热，倒入油，待油热透后倒入红辣椒丝炒几下即倒入柿子椒丝同炒，炒至颜色变深绿，即加入盐和白糖炒几下加入味精，炒匀即出锅。

## 附注

1. 炒本菜用的柿子椒应选用个头大而且肉厚的。炒时火要旺，速度要快，不宜炒得太烂。
2. 如用干红辣椒，应事先泡软。
