# 炒韭菜

## 原料

| 名称 | 重量   | 名称 | 重量 |
| ---- | ------ | ---- | ---- |
| 韭菜 | 500 克 | 精盐 | 适量 |
| 食油 | 30 克  |      |      |

## 制作方法

1. 把韭菜理去老叶，茎衣、花和老梗等，洗净，切成约 3 厘米长的段。
2. 将炒锅置火上烧热，倒入食油，又热后即把韭菜段倒入煸炒（动作要快），加精盐，炒熟迅速起锅。

## 附注

1. 韭菜在我国北方都用来做饺子或包子的馅料。而在南方都炒做菜吃。
2. 炒韭菜时一直煸炒至熟即出锅，不要盖锅盖。若盖了锅盖煮，容易发黄烂熟，失去良好风味。
3. 炒韭菜时可再加些配料，如豆腐干丝或百叶丝、绿豆芽、小虾、肉丝等。需要时也可稍加些水。使用时有的欢喜加醋或拌入蛋皮丝。
4. 韭菜也很适于煮汤，如韭菜豆腐汤、韭菜血汤（或红白汤，即用豆腐和猪血或鸡、鸭同烧）、韭菜蛋汤、韭菜粉皮肉末汤等。
